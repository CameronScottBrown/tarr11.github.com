There’s a lot of noise about teaching kids to code.

People act like it’s like a basic skill, like reading or writing.

But it’s not, it’s a way for kids to express their creativity.

It’s a combination between a team and class.

Teaching kids to code is not a vocational activity, it’s a creative activity.

I’m not teaching kids to code so that they can get a job. I’m teaching them so that they can express their creativity.
